Getting Started

Downloading

To start using this CSS Responsive Framework, simply download the compressed .zip version or clone the GitHub repository to your desktop.

$ git clone https://github.com/christiangibbs/css-responsive-framework.git

Setting it up

Once you have a copy of the framework, you need to reference the styleshseet and script (optional) in the <head> of all files you wish to use the framework.

<head> <link rel="stylesheet" type="text/css" href="css-responsive-framework/style.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.0/jquery.min.js"></script> <script src="css-responsive-framework/actions.js"></script> </head>

Requirements

The only requirement for this framework is jQuery, however this is for only the extended features such as interactive elements.
